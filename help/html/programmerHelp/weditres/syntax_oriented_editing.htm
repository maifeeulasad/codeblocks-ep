Syntax oriented editing

Wedit is aware of the syntax of C programs, and uses this for:

  • Building a function index dynamically        F12
  • Checking where an identifier is used.        Ctrl+F8
  • Go to definition                                        F8
  • Matching parentheses, brackets, #if / #endif. F11
  • Identifier completion                                ESC
  • Showing inactive #ifdefs

and many others.
