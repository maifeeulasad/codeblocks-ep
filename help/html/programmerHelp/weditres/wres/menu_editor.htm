The menu editor

The menu editor shows you the menu in the form of a tree. Each top level pop-up corresponds to the top level items of the menu being edited.

To add a menu item just type insert (the ‘ins’ key), selecting before the menu item above the one you will introduce.

You do the same for introducing a new menu item in a pop-up.

You can test your menu to see it in ‘action’ by choosing ‘test’  in the ‘File’ option of the main menu of the menu editor.

Besides the tree with the items, you will se the following dialog box floating around. Here you enter the name of the menu, its associated symbol (#define), and its value.

In the ‘Options’ group you can change the appearance of the menu item, either for a normal menu item, or for a popup menu. Do not forget to press ‘Apply’ when you are done with changes.
