Opening/Deleting existing projects

This dialog prompts you for a project that you will open (or delete from the list). When the selection changes from one project to the other you can see in the text box more information about the project selected.

If you check the checkbox "Open in a new window", Wedit will remain open and a new executable will start with the new project.

Note the last line of the dialog box: it shows you where the list of projects is stored.
